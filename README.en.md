## Ruoyi profile



Ruoyi is a full set of open source rapid development platform, without reservation for individuals and enterprises to use free.



* Front end uses Vue, Element UI.

* The backend uses Spring Boot, Spring Security, Redis & Jwt.

* Authorization authentication using Jwt, support multi-terminal authentication system.

* Support loading dynamic permissions menu, multiple easy permissions control.

* Efficient development, the use of code generator can be a key to generate the front and back end code.



## Project Introduction

The team members are all ordinary undergraduates from a university. This project is an online examination system based on RuoYi, used for the final software engineering practice training



## Function introduction - student end

1. Students can participate in the online exam by logging in with their account and password. If the administrator has not added a student account, students can register by themselves.

2. After entering the exam system, students have multiple functional menus: online exam, my score, wrong question book, knowledge base, etc.

3. Online exam: Online exam displays the exam information created by the current administrator, including the exam name, exam type, exam date limit, exam duration limit, total score, and pass line. Students click the corresponding button to enter the preparation exam.

4. Prepare exam: The basic information of the exam is displayed on the prepare exam page. Click "Start Exam" to enter the exam interface.

5. Start exam: After the exam begins, the remaining time and paper submission will be displayed at the top of the page, the answer sheet will be displayed on the left side of the page, and questions and options will be displayed in the center of the page.

6. Results of exam: after students hand in their papers, the page displays the results of exam, from which students can learn the correct answers, selected answers, exam scores and other information.

7. My Score: Students can view their scores and corresponding details through my score interface.

8. Wrong Question Book: Students can view questions they answered incorrectly and mark them as mastered.

9. Knowledge Base: Students can access learning resources including articles, videos and attachments.


## Function introduction - Administrator side
1. The administrator has multiple main functions: exam management, question management, system setting, knowledge base management, etc.

2. Question bank management: The administrator can add question bank and delete the created question bank through question bank management.

3. Test question management: the administrator can add test questions through test question management, the types of test questions are: single choice, multiple choice, judgment, fill-in-the-blank and essay questions, and can set the answer analysis of the topic; To facilitate adding and modifying questions in batches, the system provides import and export functions. Users can add and upload questions in batches using the provided Excel template, and export existing questions in the system to a local computer.

4. Exam management: The administrator can view the details of the corresponding exam through the exam management, and can add, modify, and delete the existing exam.

5. Knowledge Base Management: The administrator can manage learning resources including articles, videos and attachments.

6. System configuration: The administrator answers questions through system configuration.

7. Role management: Administrators can view role information through role management.

8. User management: Administrators can add or delete users through user management.
